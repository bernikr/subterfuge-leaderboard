About

This is an attempt to improve the leaderboard of the mobile game Subterfuge.

I download the official leaderboard once a day to provide a history. I only started with archiving on the 25th of September this year. There is some older data avalible that was downloaded from the web archive's wayback machine. If you happen to have historical leaderboard data in any shape, I would be happy if you would get in touch with me. (mail, telegram)

If you want to support this project you can contribute on github or donate a coffee.
