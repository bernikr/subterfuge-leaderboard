About

This is an attempt to improve the leaderboard of the mobile game Subterfuge.

I download the official leaderboard once a day to provide a history. I started with archiving on the 25th of September 2021. There is some older data available that was downloaded from the web archive's wayback machine. If you happen to have historical leaderboard data in any shape, I would be happy if you would get in touch with me. (mail, telegram)

Many thanks to Alex Freeman for providing almost a year of data from the end of 2017 to the middle of 2018 that he collected for his analysis articles during that time.

If you want to support this project you can contribute on github or donate a coffee.
